PUT YOUR MAIN MENU BACKGROUND IMAGE HERE
=========================================

Name your image file: MainMenuBackground.png

The code will automatically load this image as the background for your main menu.

After adding the image:
1. Select it in Unity
2. Set Texture Type to: Sprite (2D and UI)
3. Click Apply

The image should be the full-screen background showing the kitchen scene
with the "Culinary Chaos" title and button areas.
